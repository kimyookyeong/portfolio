PROJECT 1


도서사이트 - 영풍문고


Open API


POINT COLOR



국내 도서사이트 영풍문고입니다.

모든 도서 목록들의 제목, 소개, 저자, 출판사, 정가, 표지 정보는
카카오 오픈 API를 활용하여 제작하였습니다.

6가지의 이미지와 배경색이 자동으로 바뀌는 슬라이더로
이벤트, 기획전 등의 다양한 정보들이 한눈에 들어올 수 있게
구현하였습니다.

왼쪽 측면의 사이드 배너는 사용자들에게 클릭을 유도하여
도서 홍보 효과를 더욱 증가시킬 수 있는 효과를 주었습니다.

서브페이지에서 영업점 재고 및 위치 확인과 쿠폰 및 결제혜택을
확인할 수 있는 모달을 제작하여 더 많은 정보를 편리하게 얻을 수
있도록 구현하였습니다.


PROJECT 2


홍보사이트 - Steak In


responsive


POINT COLOR



해외 스테이크 레스토랑 사이트입니다.

컴퓨터, 태블릿, 모바일 등 사용자의 화면에 맞게 최적화된
레이아웃으로 보이는 반응형 웹사이트로 제작하였습니다.

절제된 레이아웃을 사용하여 차분하고 정돈된 디자인으로
설계하였습니다.

주로 사각형 그리드와 표 형식으로 효과적인 정보 전달이
가능하도록 하였습니다.

미디어 쿼리를 사용한 반응형 웹페이지로 제작하여 폭이
좁은 화면일 경우 햄버거 메뉴를 이용할 수 있도록 하였습니다.
